# Server Manager App

Android приложение для управления сервером как Termius.

## Возможности
- SSH терминал
- VPN подключение  
- Управление базами данных
- Список серверов

## Установка APK

1. Нажмите **Fork** чтобы скопировать репозиторий к себе
2. Перейдите в **Actions** 
3. Нажмите **Run workflow** → **Run workflow**
4. Дождитесь сборки (2-3 минуты)
5. Скачайте APK из **Artifacts**

## Или скачать готовый APK

[Скачать ServerManager.apk](https://github.com/username/server-manager-ready/releases/latest/download/app-debug.apk)

## Настройка

В коде замените:
- `<username>` - ваш логин
- `<password>` - ваш пароль  
- `<server_ip>` - IP вашего сервера
